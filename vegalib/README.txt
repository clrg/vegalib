= Vexi Game Library (VeGaLib) =
Created by Charles Goodwin <charles.goodwin@gmail.com>

See the project page for documentation:
http://code.google.com/p/vegalib

== Running ==
Requires Vexi Platform 3.0 (core + widgets) which can be downloaded
from http://vexi.sourceforge.net/ - revision 3400+ is necessary.

 java -jar vexi3.jar -t example.DIR.main widgets.vexi src/ example/

Replace DIR with the name of one of the exampels in the example/
directory.
